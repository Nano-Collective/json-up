# Community

`json-up` is part of the [Nano Collective](https://nanocollective.org) — a community collective building local-first, privacy-respecting AI tooling for the community, not for profit.

## Contributing

We welcome contributions of all sizes. The [Contributing Guide](https://github.com/nano-collective/json-up/blob/main/CONTRIBUTING.md) covers the full development workflow: setup, testing, code style, commit messages, and the release process.

Quick links:

- **[Open issues](https://github.com/nano-collective/json-up/issues)** — pick one up or file a new one.
- **[Pull requests](https://github.com/nano-collective/json-up/pulls)** — see what's in flight.
- **[Code of Conduct](https://github.com/nano-collective/json-up/blob/main/CODE_OF_CONDUCT.md)** — read before participating.

## Discord

Real-time discussion, support, and roadmap chat happens on the [Nano Collective Discord](https://discord.gg/ktPDV6rekE). It is the fastest way to ask a question or chat with maintainers and other users.

## Reporting issues

For bugs, please use the [bug report template](https://github.com/nano-collective/json-up/issues/new?template=bug_report.md) so we have everything we need to reproduce and fix.

For feature requests, the [feature request template](https://github.com/nano-collective/json-up/issues/new?template=feature_request.md) helps frame the conversation.
